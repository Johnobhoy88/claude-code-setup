# Highland AI - Claude Code Setup Tool

## What This Project Is

A Micro-SaaS CLI tool that configures Claude Code in 5 minutes. Users pay $2.99-$4.99/month, answer an AI questionnaire, and receive a complete setup: MCPs, skills, plugins, rules, hooks, and CLAUDE.md.

**The goal:** Claude should feel like it's working *with* the developer from the first moment.

## How the Product Works

1. User visits website, picks tier (Basic/Pro)
2. AI questionnaire captures project details
3. Payment via Stripe
4. Backend generates config (MCPs, skills, rules, CLAUDE.md)
5. User receives email with token
6. User runs: `npx @highland-ai/claude-setup --token XXX`
7. CLI installs everything
8. (Pro) Claude does final Q&A on first startup
