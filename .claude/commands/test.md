# Run Tests

Run the test suite for the specified component.

## Usage
/test [component]

Components: cli, backend, frontend, all

## Instructions

Run the appropriate test command:

- **cli**: `cd cli && npm test`
- **backend**: `cd backend && npm test`
- **frontend**: `cd frontend && npm test`
- **all**: Run all three sequentially

Report results clearly with pass/fail counts.

If tests fail, analyze the failures and suggest fixes.
