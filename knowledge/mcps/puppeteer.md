# Puppeteer MCP

## Overview

Puppeteer MCP provides browser automation capabilities for web scraping, testing, and screenshot capture. It enables Claude to interact with web pages programmatically.

## Package

`@anthropic/puppeteer-mcp`

## Use Cases

- Web scraping and data extraction
- Browser-based testing
- Screenshot capture for debugging
- Form automation and testing
- Visual regression testing

## Configuration

```json
{
  "puppeteer": {
    "command": "npx",
    "args": ["-y", "@anthropic/puppeteer-mcp"]
  }
}
```

## Best Practices

1. Use for visual debugging of web apps
2. Great for testing user flows
3. Capture screenshots to verify UI changes
4. Automate repetitive browser tasks

## When to Recommend

- Web development projects
- Frontend applications
- Projects requiring web scraping
- E2E testing workflows
- UI debugging scenarios
