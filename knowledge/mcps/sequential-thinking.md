# Sequential Thinking MCP

## Overview

Sequential Thinking MCP enhances Claude's reasoning capabilities through structured step-by-step thinking. It helps break down complex problems into manageable steps.

## Package

`@anthropic/sequential-thinking-mcp`

## Use Cases

- Complex debugging scenarios
- Architecture decision making
- Multi-step problem solving
- Code refactoring planning
- System design discussions

## Configuration

```json
{
  "sequential-thinking": {
    "command": "npx",
    "args": ["-y", "@anthropic/sequential-thinking-mcp"]
  }
}
```

## Best Practices

1. Use for complex, multi-step problems
2. Great for debugging difficult issues
3. Helps with architectural decisions
4. Useful for planning large refactors

## When to Recommend

- Complex projects with intricate logic
- Debugging challenging issues
- Architecture and design work
- Projects requiring careful reasoning
- Any project where step-by-step thinking helps
