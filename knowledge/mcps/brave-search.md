# Brave Search MCP

## Overview
Brave Search MCP for privacy-focused web search queries and quick lookup tasks.

## Installation
```bash
npx @anthropic/mcp-server-brave-search
```

Configuration
- `BRAVE_API_KEY` from Brave Search API.
- Optional: set safe search or region preferences if supported.

Capabilities
- Run web search queries with concise results.
- Retrieve titles/snippets/links for quick research.
- Filter results based on query parameters.

Example Usage
- Find current docs for a library or API.
- Gather top resources for a troubleshooting topic.
- Check recent news or updates about a dependency.

Best Practices
- Use specific queries to reduce noise.
- Cite links in outputs; avoid copying large content.
- Respect API rate limits to prevent throttling.
